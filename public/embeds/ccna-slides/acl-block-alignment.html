Block-Ausrichtung

Ziel: 192.168.20.46 – .95
ZAHLENSTRAHL — Soll vs. was der Router wirklich trifft permit 192.168.20.46 0.0.0.31
▮ gewünscht ▮ tatsächlich (zu viel .32–.45, es fehlt .80–.95 aus dieser Zeile)
WARUM — die „egal“-Bits
Adresse 46 001 0 1110 ← 5 egal-Bits (Wildcard 31)
Router nullt sie 001 0 0000 = 32
Die gesetzten Wildcard-Bits darf der Router ignorieren — er behandelt sie als 0. Deine .46 wird dadurch zu .32: dem nächsten Vielfachen von 32 darunter.
DIE PROBE
Blockgröße = Wildcard + 1
0.0.0.31 → 31 + 1 = 32
Start mod Blockgröße = 0 ?
46 mod 32 = 14  ✗ nicht ausgerichtet
64 mod 32 = 0   ✓ ausgerichtet
Gültige 32er-Starts: 0, 32, 64, 96, 128, …
