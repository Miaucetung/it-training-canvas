WAS IST ROUTING?

Router entscheidet anhand der Routing-Tabelle, über welchen Ausgang ein Paket weitergeleitet wird.

→ Connected — direkt verbunden · AD 0
→ Static — manuell konfiguriert · AD 1
→ Dynamic — per Protokoll gelernt
STATISCH ✓
+ Volle Kontrolle
+ Kein BW-Overhead
+ Kein Update-Traffic
− Kein automatischer Failover
− Manuell skalierbar
DYNAMISCH ✓
+ Automatisch anpassbar
+ Redundanz / Failover
+ Großnetze skalierbar
− CPU / Bandbreite
− Komplexer
