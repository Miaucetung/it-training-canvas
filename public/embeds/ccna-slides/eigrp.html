ADV. DISTANCE VECTOR

EIGRP — Enhanced Interior Gateway Routing Protocol

DUAL-ALGORITHMUS
FD — Feasible Distance: Gesamtmetrik zu einem Ziel
RD / AD — Reported/Advertised Distance: Metrik des Nachbarn
Successor — bester Pfad, in Routing-Tabelle installiert
Feasible Successor — Backup-Pfad (RD < FD des Successors)
DREI TABELLEN
Neighbor Table
Direkt verbundene EIGRP-Router mit Hello-Intervall
Topology Table
Alle bekannten Routen mit FD / RD-Werten
Routing Table
Nur Successor-Routen — installiert für Forwarding
COMPOSITE METRIC
256 × (K1×BW + K3×Delay)
BW = 10⁷ / min. Bandbreite (kbps)
Delay = Summe Delays / 10 (µs)
Default: K1=1, K3=1, K2=K4=K5=0
DUAL · SUCCESSOR vs. FEASIBLE SUCCESSOR
KONFIGURATION
! Classic Mode
router eigrp 100
network 10.0.0.0 0.0.0.255
no auto-summary
passive-interface g0/1
! Bandbreite anpassen
interface g0/0
bandwidth 1000000
! Router-ID setzen
eigrp router-id 1.1.1.1
! Verifizierung
show ip eigrp neighbors
show ip eigrp topology
show ip route eigrp
show ip eigrp interfaces
